# Firefox + Marionette + geckodriver

## Prerequisites

- [Firefox Developer Edition](https://www.mozilla.org/en-US/firefox/developer/)
- [geckodriver](https://github.com/mozilla/geckodriver/)

## Installation

    npm install

## Usage

    node index.js
